Hi，这是我的中文博客，为什么它的页面看起来这么素（简约）？

大概是某天，我打开我之前用 Hexo + Next 整的博客，看着它慢悠悠的打开，加载一堆 JavaScript，而且还需要等待页面动画结束。

我开始思考，真的有必要这么复杂吗，刚好那天看到了 Bear Blog，受它启发，我直接自己用 Hugo + Simple.css 整了个模板。也就是现在你看到的网页，它真的非常非常快，没有任何 JavaScript。依赖于 Simple.css，我在实现时几乎不用留意样式，只需要使用正确的 HTML 标签，整个网站的自定义 CSS 只有 3 处。
